Readme bij Victorie Boogie Woogie in Scratch
------------------------------------------------------
Gemaakt door: Sergio van Pul
Met dank aan: Joek van Montfort

Deze versie van Victory Boogie Woogie is gemaakt met de educatieve software Scratch.
Het is nodig de Scratch applicatie te downloaden om het programma te openen en te bekijken.
De Scratch software is te downloaden via de volgende link:
www.scratch.mit.edu/download

Omdat Scratch een leermiddel is zijn code en presentatie niet van elkaar gescheiden, maar worden ze samen getoond.
Het is zelfs mogelijk om aanpassingen te maken terwijl het programma draait.

In deze versie is niet zozeer het eindresultaat belangrijk, maar het proces waarmee het beeld wordt opgebouwd. Kleurvlakken worden neergezet en overschreven, waardoor een constant veranderend landschap ontstaat. De weergave benadrukt het ritme en de beweging die in het werk zichtbaar is.


Instructies:
--------------
spatiebalk =
schilder een nieuwe Boogie Woogie met wachttijd, zodat je het proces goed kan volgen.

t = 
schilder een nieuwe Boogie Woogie zonder wachttijd, wanneer je geen geduld hebt. De opbouw van het werk kost nog steeds enige tijd.

Rechtsklik op het schilderij en kies 'afbeelding van huidige scherm opslaan om een screenshot te bewaren.

Klik op het 'presentatiemodus' icoontje in de rechter bovenhoek om het canvas op het hele scherm te tonen. Met het 'return' icoontje links boven de weergave kun je terug gaan naar de ontwikkelomgeving.

De code wordt nader toegelicht in het programma.
